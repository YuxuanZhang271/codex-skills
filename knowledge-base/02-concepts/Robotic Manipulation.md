# Robotic Manipulation

Robotic manipulation covers perception, planning, control, and learning methods for robots that physically interact with objects.

## Dimensions

- Task: reaching, pushing, grasping, pick-and-place, insertion, tool use.
- Robot state: joint positions, velocities, end-effector pose, gripper state.
- Observation: RGB, depth, tactile, force-torque, proprioception.
- Action: joint control, Cartesian control, waypoints, skills, learned policy outputs.
